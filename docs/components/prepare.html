开发准备

技能准备

开发自定义组件，你需要掌握以下技能：

  • 熟悉千帆 ApaaS 的产品的基础使用
  • 熟悉前端 Web 开发的基础知识，了解 Javascript, 了解React开发；
  • 熟悉终端的使用，了解如何在CMD、Shell中运行命令；

环境准备

开发自定义组件，你需要准备：

  • 安装 nodejs;
  • 安装一个前端 IDE, 推荐安装VSCODE;
